## Project Overview
The Language Translation Tool is a simple web-based application that allows users to translate text from English into multiple languages. The project is developed using HTML, CSS, and JavaScript and uses a public translation API for real-time translation.

This project is created as part of an internship task to demonstrate frontend development skills and API integration.

## Features
- Translate text from English to:
  - Hindi
  - Urdu
  - French
  - Spanish
  - Chinese
  - Korean
- Clean and responsive user interface
- Gradient-based professional UI design
- Copy translated text with one click
- Text-to-Speech functionality
- No API key required

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- MyMemory Translation API
- Web Speech API (for Text-to-Speech)

## How It Works
1. User enters text in the input box.
2. User selects the target language.
3. On clicking the **Translate** button, the text is sent to the translation API.
4. The translated text is displayed on the screen.
5. The user can copy the translated text or listen to it using text-to-speech.

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` in any modern browser  
   **OR**  
   Use Live Server extension for best performance.
4. Enter text and start translating.
